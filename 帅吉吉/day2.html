    # 超链接的四种状态

    <!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style>
			a:link{
				/* 当超链接没有被访问过时的样式 */
				color: green;
			}
			a:visited{
				/* 当超链接已经被访问过时的样式 */
				color: #000;
			}
			/* hover - 重点 */
			a:hover{
				/* 当鼠标移动到超链接上时的样式 */
				color: pink;
			}
			a:active{
				/* 当超链接被选中时的样式 */
				color: red;
			}
			/* 如果四种状态需要同时设置的话，那么lvha的顺序不能改变 */
		</style>
	</head>
	<body>
		<a href="https://le.ouchn.cn/home">跳转</a>
	</body>
</html>

    # 浮动

    <!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style>
			*{
				margin: 0;
				padding: 0;
			}
			ul{
				list-style: none;
			}
			li{
				width: 200px;
				height: 200px;
			}
			.a1{
				background-color: red;
				/* 浮动 - 网页布局*/
				/* 特点
					1. 浮动的元素会脱离文档流，不占据页面空间，后续的元素会上前补位
					2. 浮动的元素会向左或者向右浮动，取决于取值为left还是right
					3. 浮动的块级元素会失去独占一行的特点
					4. 如果浮动的元素前已经有了一个已经浮动的元素，则这个元素会停靠在前面浮动元素的旁边
					5. 浮动专门用来解决块级元素在一行显示的问题
					6. 浮动的元素依然位于父元素之内
				 */
				/* float: none/left/right; */
				float: left;
			}
			.a2{
				background-color: green;
				float: left;
			}
			.a3{
				background-color: blue;
				float: left;
			}
		</style>
	</head>
	<body>
		<ul>
			<li class="a1"></li>
			<li class="a2"></li>
			<li class="a3"></li>
		</ul>
	</body>
</html>
